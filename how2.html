How I created my To-Do list and Calculator

Calculator

In order to create a calculator you need to be able to work with javascript, and html So what I did when creating the calculator you have to make input bottons so everything inside the calculator's boxes are being inputed to do a certain task. Like you enter a number in a box and choose what type of math you would like to use to solve it, and then you have it solve your problem. In java script you have to create if else statements so the code will know what to do if a certain task was given to them, like subtracting a centain number from the other. This is what the input code looks like.(In Javascript) 
var input1 = document.getElementById("input1");
var input2 = document.getElementById("input2");
var button = document.getElementById("calculate");
var dropbox = document.getElementById("dropbox");
var answer = document.getElementById("answer");
var result;

button.addEventListener("click",

function (){
    if (dropbox.value == "-"){
        result = parseInt(input1.value) - parseInt(input2.value);
        answer.innerHTML = result;
    }else if (dropbox.value == "+"){
        result = parseInt(input1.value) + parseInt(input2.value);
        answer.innerHTML = result;
    }else if (dropbox.value == "/"){
        result = parseInt(input1.value) / parseInt(input2.value);
        answer.innerHTML = result;
    }else if (dropbox.value == "*"){
        result = parseInt(input1.value) * parseInt(input2.value);
        answer.innerHTML = result;
    }
})
